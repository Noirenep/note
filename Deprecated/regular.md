## 正则表达式
### 语法规则
1. 行定位符  
 "^"表示行的开始  
 "$"表示行的结尾  

2. 单词定界符 `\b \B`   
 查找一个完整的单词`\bword\b`  

3. 正则表达式区分大小写   

4. 选择字符 "|"  

5. 连字符 "-"  
 `[a-zA-Z]`

6. 排除字符 "^"  

    `[^1-5]`  
7. 限定符
```
*   零次或多次   等价于{0,}
+   一次或多次   等价于{1,}
?   零次或一次   等价于{0,1}
{n} 匹配N次
{n,}至少n次
{n,m} n到m次
```

8. 点字符"." 匹配除了换行符以外的任意一个字符  

9. 转义字符
`. ? \ ( ) $ + [ ] ^& |`  
加`"\"`后转义  

10. `"\"`
```bash
    POSIX                   PCRE
[[:digit:]]  任何数字        \d
[[:alnum:]]  任何数字字母    \w
[[:alpha:]]  任何字母
[[:blank:]]  任何空白
[[:upper:]]  所有大写字母
[[:lower:]]  所有小写字母
[[:space:]]  空白字符        \s  相当于[\f\n\r\t\v]  \S非空白
```
11. 圆括号
    1. 改变作用范围 `|*^`  例如:`(my|your)baby`
    2. 分组(\.[0-9]{1,3}){3}
