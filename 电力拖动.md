# 电力拖动

## 一。绪论
1. 电力拖动实现了电能与机械能之间的转换
2. 控制电动机电压电流频率，改变机械的转矩，速度位移等机械量
3. 交流调速系统取代直流调速系统
    - 直流电动机数学模型简单，转矩易于控制
    - 交流电动机结构简单，蛋动态数学模型具有非线性多变量强耦合的性质，比直流电动机复杂
    - 同步电动机的转速与电源频率严格保持同步，机械特性硬
4.  转矩控制是运动控制的根本问题，磁链控制与转矩控制同样重要。
